## How to Capture Screenshots

### On Windows:
1. **Snipping Tool** - Search for "Snipping Tool" in Start menu
2. **Snip & Sketch** - Windows + Shift + S
3. **Firefox Screenshot Tool** - Right-click page → "Take Screenshot"

### On macOS:
1. **Command + Shift + 4** - Select area to capture
2. **Command + Shift + 3** - Full screen capture
3. **Firefox Screenshot Tool** - Right-click page → "Take Screenshot"

### On Linux:
1. **Screenshot tool** - Varies by distro (GNOME Screenshot, KDE Spectacle, etc.)
2. **Firefox Screenshot Tool** - Right-click page → "Take Screenshot"

## Tips for Good Screenshots

- **Use a clean window** - Close unnecessary tabs/windows in the background
- **Full page context** - Include the browser chrome (address bar) for context
- **Good contrast** - Use the default Purelymail theme
- **High resolution** - At least 1280x800, preferably 1920x1080
- **No personal data** - Use test/example email addresses if possible
- **Consistent sizing** - Keep all screenshots the same dimensions

## Firefox Add-on Store Requirements

From Mozilla's guidelines:

- **Format:** PNG, JPEG, or GIF
- **Recommended size:** 1280x800 or larger
- **Maximum file size:** 5 MB per image
- **Maximum number:** 10 images
- **Aspect ratio:** 16:9 or 4:3 recommended

## After Capturing

1. Name files according to the scheme above (01-before.png, etc.)
2. Save them to this directory
3. Compress/optimize images if needed (use tools like TinyPNG)
4. Upload to Firefox Add-on submission form in order
5. Add captions in the submission form:
   - "Before: Default Purelymail user list"
   - "After: Organized by domain with search"
   - "Search feature with real-time filtering"
   - "Collapsed view for clean overview"
